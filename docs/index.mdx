# Tutoriel Git

Ce site web est un tutoriel pour apprendre à utiliser Git. Les premières pages
donnent une explication du fonctionnement de Git, puis une série d'exercices
thématiques feront pratiquer les cas d'utilisation les plus courants.

## Contribuer

Pour contribuer (correction typographique, clarification, exercice, nouvelle section), ouvrez
simplement un [billet](https://github.com/archambaultv-prof/git/issues/new)
(issue) ou une [demande de fusion (pull request)](https://github.com/archambaultv-prof/git/pulls).
Toute contribution implique l'acceptation des licences du site (voir
ci-dessous) – *merci de faire vivre ce matériel d'apprentissage !*
