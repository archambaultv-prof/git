# Tutoriel Git

Ce site web est un tutoriel pour apprendre à utiliser Git. Il est destiné aux
étudiants du [Collège de Maisonneuve](https://www.cmaisonneuve.qc.ca) et enseigne
les cas les plus courants de l'utilisation de Git par les étudiants dans le cadre de
leurs cours.

Le tutoriel est divisé par cas d'utilisation et non par commandes Git. Chaque
section présente un cas d'utilisation et les commandes Git associées.
