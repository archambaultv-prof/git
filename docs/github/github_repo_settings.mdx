# Règlages du dépôt GitHub

## Collaborateurs

Si vous travaillez en équipe, il est possible d'ajouter les autres membres de
l'équipe comme collaborateurs du dépôt. Ceci est aussi nécessaire si vous
voulez un dépôt privé que le professeur doit pouvoir consulter. Allez dans
l'onglet `Settings` du dépôt, puis dans `Collaborators`. Vous pouvez alors
ajouter les autres membres avec leur nom d'utilisateur GitHub.
